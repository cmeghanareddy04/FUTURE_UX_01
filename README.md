# ☕ Latte Love Coffee Café -

## Overview

**Latte Love Coffee Café** is a modern coffee shop mobile application designed to provide users with a seamless and enjoyable coffee ordering experience. This project was completed as part of my **UI/UX Design Internship**, focusing on creating an intuitive, visually appealing, and user-centered interface.

The design emphasizes simplicity, accessibility, and a warm café-inspired aesthetic while enabling users to browse the menu, customize drinks, place orders, and make secure payments effortlessly.

---

## Problem Statement

Many coffee ordering applications overwhelm users with cluttered interfaces and complicated ordering processes. The objective of this project was to design a clean, intuitive, and engaging mobile experience that allows customers to discover products, customize orders, and complete purchases quickly.

---

## Objectives

* Design a visually appealing coffee café application.
* Create a simple and intuitive user experience.
* Improve navigation and accessibility.
* Streamline the ordering and checkout process.
* Build a consistent design system with reusable components.

---

## Design Process

The project followed a user-centered design approach:

1. User Research
2. Competitor Analysis
3. User Flow Creation
4. Low-Fidelity Wireframes
5. High-Fidelity UI Design
6. Interactive Prototype
7. Design Evaluation & Iteration
