LATTE LOVE COFFEE CAFÉ
UI/UX Internship Project (FUTURE_UX_01)
=======================================

OVERVIEW

Latte Love Coffee Café is a modern coffee shop mobile application designed to provide users with a seamless and enjoyable coffee ordering experience.

This project was completed as part of my UI/UX Design Internship (FUTURE_UX_01). The application focuses on creating an intuitive, visually appealing, and user-centered interface that enables customers to browse coffee categories, view product details, customize drinks, manage their cart, and complete secure payments effortlessly.

The design combines modern UI principles with a warm café-inspired color palette to deliver an engaging digital coffee experience.

---

PROBLEM STATEMENT

Many coffee ordering applications have cluttered interfaces and complicated ordering processes. This project aims to simplify the customer journey by providing a clean, intuitive, and visually appealing mobile application that makes browsing, ordering, and checkout quick and effortless.

---

OBJECTIVES

• Design a modern and visually appealing coffee ordering application.
• Create an intuitive and user-friendly experience.
• Improve navigation and accessibility.
• Simplify the ordering and checkout process.
• Build a consistent and reusable design system.
